# Dawit Tadele's Personal Portfolio

![Portfolio Preview](https://via.placeholder.com/800x400/0d6efd/ffffff?text=Dawit+Tadele+Portfolio+Preview)

A clean, modern, and fully responsive multi-page personal portfolio website built from scratch using **HTML5, CSS3, and vanilla JavaScript**.

**Live Demo:** [View Portfolio](https://dawit-gt.github.io/dawit-portfolio/)

## ✨ Features

- **Multi-Page Structure** - Dedicated pages for Home, About, Projects, and Contact
- **Fully Responsive Design** - Optimized for all devices (mobile-first approach)
- **Dark/Light Mode Toggle** - Smooth theme switching with persistent preference
- **Typing Animation** - Dynamic hero section with rotating text
- **Smooth Scrolling Navigation** - Seamless page navigation
- **Interactive Mobile Menu** - Hamburger menu for mobile devices
- **Animated Skill Progress Bars** - Visual representation of technical skills
- **Working Contact Form** - Ready for backend integration (currently demo)
- **Modern Card-Based Layout** - Clean project showcase
- **Professional UI/UX** - Attention to detail and user experience

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and accessibility
- **CSS3** - Flexbox, Grid, CSS Variables, Animations, Responsive design
- **Vanilla JavaScript** - DOM manipulation, event handling, local storage
- **Font Awesome** - Icon library for visual elements
- **GitHub Pages** - Hosting and deployment

## 🚀 Getting Started

### Prerequisites

- A modern web browser
- No installation required - pure HTML/CSS/JS

### Running Locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/dawit-gt/dawit-portfolio.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd dawit-portfolio
   ```

3. **Open in browser:**
   - Open `index.html` directly in your browser
   - Or use a local server (recommended for full functionality):

     ```bash
     # Using Python
     python -m http.server 8000

     # Using Node.js
     npx serve .
     ```

4. **Navigate the site:**
   - **Home** (`index.html`): Hero section and introduction
   - **About** (`about.html`): Detailed about me and skills
   - **Projects** (`projects.html`): Featured project showcase
   - **Contact** (`contact.html`): Contact form

## 🎨 Customization

### Changing Content

- Edit `index.html` to update personal information, projects, and contact details
- Modify the `texts` array in `js/script.js` for different typing animations

### Styling

- Update CSS variables in `css/styles.css` for color scheme changes
- Adjust breakpoints in the responsive media queries

### Adding Projects

- Duplicate project card HTML in `index.html`
- Update image sources and links accordingly
